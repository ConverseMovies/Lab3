lab three

The Lab 3 project is to create a portfolio website for our Senior Design group to show off our group and what we have done. The website also includes a contact form and some password-protected content.


Key features include website creation through HTML, CSS, and JavaScript from scratch. Our design also implements Python's Flask for local hosting and advanced password functionality. The website is hosted through Heroku's available services and ease of integration with Flask's interface. There is also an admin-only Python program for changing the password to protected content.
