Lab 1: ESP32 Integrated System

Project Summary

The Lab 1 project successfully integrated an ESP32 microcontroller with sensors, an LCD display, and a Java GUI to monitor temperature data in real-time. Utilizing Firebase Realtime Database, the system enables cloud-based communication between hardware and software components.

Key features include user-defined temperature thresholds, automated email notifications, and graphical data visualization. Initially implemented in C++, challenges with GUI and HTTP libraries prompted a switch to Java, leveraging Swing and JFreeChart for robust interface and graph functionality.

Highlights:

  • ESP32 microcontroller connected to sensors and LCD.
  • Java GUI with real-time graph plotting and email alerts.
  • Cloud communication via Firebase.
